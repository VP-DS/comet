# PrettyBytes

<Meta title="docs/@comet/admin/helpers/prettyBytes" />

The PrettyBytes component can be used to display byte values e.g. file sizes.

Supported units are: `"byte" | "kilobyte" | "megabyte" | "gigabyte" | "terabyte" | "petabyte"`
