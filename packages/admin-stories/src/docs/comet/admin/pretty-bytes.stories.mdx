# PrettyBytes

<Meta title="docs/@comet/admin/helpers/prettyBytes" />

The PrettyBytes component can be used to display file sizes or other byte values.

Supported units are: `"byte" | "kilobyte" | "megabyte" | "gigabyte" | "terabyte" | "petabyte"`

<Canvas>
    <Story name="PrettyBytes - Normal Behavior">
        {() => {
            const Story = () => {
                return (
                    <div style={{ display: "grid", gridTemplateColumns: "auto auto", gap: "10px" }}>
                        <span>6.000 Bytes</span>{" "}
                        <span>
                            <PrettyBytes value={6000} />{" "}
                        </span>
                        <span>6.000.000 Bytes</span> <span>
                            <PrettyBytes value={6000000} />{" "}
                        </span>
                    </div>
                );
            };
            return <Story />;
        }}
    </Story>
</Canvas>

## Fixed Unit

Per default, the unit is determined automatically. But it is also possible to manually set the unit. In the following example, the unit is manually set to `kilobyte`.

<Canvas>
    <Story name="PrettyBytes - Fixed Unit">
        {() => {
            const Story = () => {
                return (
                    <div style={{ display: "grid", gridTemplateColumns: "auto auto", gap: "10px" }}>
                        <span>6.000.000 Bytes</span>{" "}
                        <span>
                            <PrettyBytes value={6000000} unit="kilobyte" />{" "}
                        </span>
                    </div>
                );
            };
            return <Story />;
        }}
    </Story>
</Canvas>

## Custom Maximum Fraction Digits

The default value for `maximumFractionDigits` is `2`. But it is possible to override this value. In the following example, the maximumFractionDigits are set to `4`.

<Canvas>
    <Story name="PrettyBytes - Custom Maximum Fraction Digits">
        {() => {
            const Story = () => {
                return (
                    <div style={{ display: "grid", gridTemplateColumns: "auto auto", gap: "10px" }}>
                        <span>6.000 Bytes</span>{" "}
                        <span>
                            <PrettyBytes value={6000} unit="megabyte" maximumFractionDigits={4} />{" "}
                        </span>
                    </div>
                );
            };
            return <Story />;
        }}
    </Story>
</Canvas>
