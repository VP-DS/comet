# FieldContainer

`FieldContainer` is used to contain an input and provide basic styling/layout within the form.  
It is also used to set a label, error--messages and other common form-field attributes.

_Note: If your form is built with final-form, use [Field](/docs/docs-form-components-field--page) instead._

## Props

| Name           | Type                                         | Default value   | Description                                                                                                                                                                                                      |
| :------------- | :------------------------------------------- | :-------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --- |
| children       | `node`                                       |                 | The fields input component.                                                                                                                                                                                      |
| label          | `string` \| `node`                           |                 | The label of the input.                                                                                                                                                                                          |
| fullWidth      | `boolean`                                    | false           | If `true` the field will be full-width. _See [Form Layout](/docs/docs-form-layout--page) for details and examples._                                                                                              |
| variant        | `"vertical"` \| `"horizontal"`               | "vertical"      | If set to `vertical`, the label will be above the input. If set to `horizontal`, the label and the input will be next to each other. _See [Form Layout](/docs/docs-form-layout--page) for details and examples._ |
| fieldMargin    | `"always"` \| `"never"` \| `"onlyIfNotLast"` | "onlyIfNotLast" | Under what condition should a margin be added to the FieldContainer.                                                                                                                                             |     |
| required       | `boolean`                                    | false           | If `true`, the field will be required.                                                                                                                                                                           |
| requiredSymbol | `string` \| `node`                           | "\*"            | The symbol used to mark a field as required.                                                                                                                                                                     |
| disabled       | `boolean`                                    | false           | If `true`, the field will be disabled.                                                                                                                                                                           |
| error          | `string`                                     |                 | The error-message that, if defined, will be shown below the field and mark it as an error.                                                                                                                       |
| warning        | `string`                                     |                 | The warning-message that, if defined, will be shown below the field and mark it as a warning.                                                                                                                    |
| scrollTo       | `boolean`                                    |                 | When changed to `true`, the field will be scrolled into view.                                                                                                                                                    |
